# Paradise Nursery - e-plantShopping

Welcome to **Paradise Nursery**, an e-commerce React application designed for plant enthusiasts! This project is managed under the repository name **e-plantShopping**.

## 🌿 Project Overview
Paradise Nursery is a vibrant, user-friendly online store where users can browse an exquisite collection of indoor and outdoor plants. The application provides a seamless shopping experience, allowing users to explore different plant categories, view pricing, and manage their gardening selections effortlessly.

## ✨ Key Features
* **Landing Page:** An elegant welcome screen featuring a "Get Started" button to transition smoothly into the store.
* **Product Catalog:** A responsive grid showcasing various plants with detailed information, images, and pricing.
* **Interactive Shopping Cart:** Easily add plants to the cart, adjust item quantities dynamically, or remove items, with real-time total cost updates.
* **Responsive Navigation:** A clean navbar allowing users to jump between the home landing page, plant listings, and their current cart summary.

## 🛠️ Tech Stack
* **Frontend:** React.js, HTML5, CSS3
* **Build Tool:** Vite
* **Version Control:** Git & GitHub# coding-project-template
